# BlueConnect
![https://github.com/grainiac/BlueConnect/BlueConnect.iconset/icon_512x512.png](https://github.com/grainiac/BlueConnect/blob/main/BlueConnect.iconset/icon_128x128.png)

**BlueConnect** is a MacOS helper app that directly opens the Bluetooth Configuration otherwise only accessible from Apple's AUDIO-MIDI Setup app. The screenshot shows the app window that allows to connect and disconnect Bluetooth MIDI devices. In this case a Kawai NV10 hybrid piano:
